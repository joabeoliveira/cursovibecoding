# 🎧 Curso Vibe Coding – Serviço Público

Bem-vindo ao repositório oficial do **Curso Vibe Coding** voltado para quem trabalha no **serviço público** (servidores, terceirizados, estagiários e colaboradores de órgãos públicos).

O foco é aprender, na prática, a construir interfaces Web simples e automações leves usando:

- **HTML5**
- **CSS3 + Tailwind CSS (via CDN)**
- **JavaScript ES6+ (vanilla, sem frameworks pesados)**
- **n8n** para automações e webhooks
- Deploy simples em **VPS com painel** (EasyPanel / Coolify)

Tudo com atenção especial a acessibilidade, segurança e contexto gov.

---

## 📚 Estrutura do Curso

Este repositório será organizado por **aulas**, cada uma em sua própria pasta, com materiais interativos que rodam direto no navegador (sem necessidade de build ou instalação de Node.js).

Por exemplo:

- `aula01-setup/` – preparação do ambiente (VS Code, extensões, Git/GitHub opcional, repositório de skills)
- `aula02-.../` – HTML + Tailwind na prática
- `aula03-.../` – JavaScript interativo
- `aula04-.../` – VPS + painel
- `aula05-.../` – n8n e automações

Conforme o curso evoluir, novas pastas de aula serão adicionadas aqui.

---

## 🧩 Aula 1 – Setup (já disponível)

A primeira aula já está disponível em formato de página interativa:

- Pasta: `aula01-setup/`
- Arquivo principal: `aula01-setup/index.html`

### O que você faz na Aula 1

Ao abrir `aula01-setup/index.html` no navegador, você terá:

- Explicação do objetivo do curso e do contexto de uso no serviço público.
- Passo a passo para:
  - Instalar o **Visual Studio Code**.
  - Configurar extensões essenciais (Live Server, HTML/CSS, Tailwind, formatter etc.).
  - Entender o papel de **Git** e **GitHub** (opcional, mas recomendado).
- Um **checklist interativo** para acompanhar o que você já concluiu.
- Um **quiz rápido** para reforçar conceitos (VS Code, GitHub, repositório de skills).
- Uma **barra de progresso** indicando quanto da aula você já completou.

### Como usar

1. **Faça o download ou clone** este repositório.
2. Abra a pasta em seu editor (VS Code recomendado).
3. Localize a pasta `aula01-setup/`.
4. Abra o arquivo `index.html` no navegador:
   - Método simples: clique duas vezes no arquivo.
   - Método recomendado: use a extensão **Live Server** no VS Code.

Além deste repositório de aulas, utilizaremos como base o seguinte kit de skills em HTML/CSS/JS:

## 👥 Para quem é este curso?

- Pessoas que trabalham em órgãos públicos e querem:
  - Prototipar interfaces Web simples.
  - Automatizar tarefas repetitivas com n8n.
  - Entender melhor como aplicações Web funcionam, mesmo sem ser da área de TI.
- Equipes de inovação, laboratórios de gov tech e times de tecnologia interessados em **MVPs rápidos**.

Não é necessário ter experiência prévia em programação – a ideia é começar do zero, com exemplos bem guiados.
