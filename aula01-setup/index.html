Boas-vindas ao Vibe Code para o serviço público

Nesta aula você vai preparar o ambiente para desenvolver interfaces Web simples e automações com foco em uso dentro do setor público, usando apenas HTML, CSS, Tailwind CSS e JavaScript.

Ao final desta aula você será capaz de:

  • Instalar e abrir o VS Code no Windows.
  • Configurar extensões essenciais para HTML, CSS, Tailwind e JavaScript.
  • Conhecer o repositório oficial de skills do curso.
  • Criar a pasta do curso na sua máquina.
  • Acompanhar seu próprio progresso por meio desta página interativa.

Público-alvo e contexto:

  • Servidores públicos, terceirizados e estagiários que atuam em órgãos públicos.
  • Ambiente com possíveis restrições de instalação e acesso à internet.
  • Foco em soluções simples que podem rodar em VPS com painel (EasyPanel/Coolify).
  • Materiais alinhados com acessibilidade e boas práticas para gov.

Pré-requisitos ideais

Não é preciso ser da área de TI. Basta ter curiosidade e um computador com Windows.

  • Computador com Windows 10 ou superior.
  • Acesso à internet para baixar VS Code e acessar o GitHub.
  • Algum navegador moderno (Edge, Chrome, Firefox, etc.).
  • Permissão para instalar programas (ou apoio da equipe de TI para instalar por você).

! Dica para quem trabalha no serviço público

Se downloads ou instalações forem bloqueados, registre um chamado para a área de TI explicando que o curso usa ferramentas amplamente adotadas (VS Code, Git) e que o código será aberto para revisão.

Passo a passo: VS Code e extensões

Siga os passos abaixo no seu computador. Marque o checklist interativo conforme for concluindo.

1. Instalar o VS Code

  1. Acesse o site oficial do Visual Studio Code.
  2. Baixe a versão para Windows.
  3. Execute o instalador e siga as opções padrão.
  4. Ao final, abra o VS Code pela primeira vez.

2. Configurar idioma e aparência (opcional)

  • Na área de extensões, procure por “Portuguese (Brazil) Language Pack”.
  • Instale o pacote para deixar a interface em português, se preferir.
  • Ajuste o zoom (Ctrl + + / Ctrl + -) para facilitar a leitura.
  • Escolha um tema de sua preferência (claro ou escuro).

3. Instalar extensões essenciais

Abra a aba de extensões (atalho típico: Ctrl + Shift + X) e procure por:

  • Live Server – para abrir arquivos HTML no navegador com auto-reload.
  • HTML CSS Support (ou similar) – ajuda com sugestão de tags e estilos.
  • Prettier (ou outro formatter) – para formatar o código automaticamente.
  • Tailwind CSS IntelliSense – para sugerir classes do Tailwind.
  • Extensão de ícones – opcional, apenas para melhorar a visualização de pastas/arquivos.

Repositório oficial de skills do curso

Vamos usar o repositório joabeoliveira/skills como kit oficial de HTML, CSS, Tailwind, JavaScript e automações com n8n.

Algumas skills que você verá ao longo do curso:
